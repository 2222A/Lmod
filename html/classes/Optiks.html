Optiks:_getValue (self, eq_arg, argIn, o, optName)
Get the value from command line (private). If the *eq\_arg* is non-nil then use it. (--foo=eq\_arg). Otherwise get the next argument and remove it from *argIn*. If the type is number then convert it. Otherwise store a string.

Parameters:

  • self Optiks object
  • eq_arg The equal arg. (i.e. --foo=eq\_arg)
  • argIn The current list of command line arguments
  • o Optiks_Option object.
  • optName the option name.
