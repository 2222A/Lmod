Module fileOps

A collection of useful file operations.

Functions

findInPath (exec, path) find the absolute path to an executable.
isDir (d) Return true if path is a directory.
isFile (fn) Return true if file exists is and is a file or link.
isExec (fn) Returns true if file is readable and executable.
dirname (path) Return the directory part of path.
extname (path) Return the extension of a file or "" if there is none.
removeExt (path) Remove extension from path.
barefilename (path) return the file name w/o any directory part.
splitFileName (path) split a path into a directory and a file.
pathJoin (...) Join argument into a path that has single slashes between directory names and no trailing slash.
mkdir_recursive (path) Create a new directory recursively.
abspath (path[, localDir]) find true path through symlinks.
path_regularize (value) Remove leading and trail spaces and extra slashes.
